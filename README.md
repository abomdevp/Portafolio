# portafolio web
Aplicación web personal que presenta mi perfil como Frontend Developer, mostrando proyectos reales, habilidades técnicas y mi forma de trabajo, con foco en buenas prácticas, accesibilidad y experiencia de usuario.

# Demo
https://abomdevp.github.io/Portafolio/

# Funcionalidades
- Presentación clara del perfil y rol profesional
- Sección About Me orientada a valor e impacto
- Listado de proyectos con descripción, tecnologías y enlaces
- Sección de habilidades técnicas y herramientas
- Diseño responsive (mobile first)
- Tema claro / oscuro
- Animaciones sutiles enfocadas en UX
- Accesibilidad WCAG 2.1 AA

# Uso
1. Clonar o descargar el proyecto
2. Abrir index.html en el navegador
3. Listo (no requiere instalar dependencias)
